# 邮件管理-基础框架

本文档随插件清单打包，面向插件市场展示。新增文档资源不会改变插件运行时行为。

## 概述

管理邮件服务器连接与邮箱账号，并为 SMTP、IMAP、POP3 等协议插件提供共用的邮件能力。系统邮件通知也经由本插件投递。

## 文档导航

- [配置说明](configuration.md)
- [更新日志](changelog.md)

## 功能亮点

- 管理邮件服务器连接与邮箱账号，并为 SMTP、IMAP、POP3 等协议插件提供共用的邮件能力。系统邮件通知也经由本插件投递。
- 提供邮件管理等工作台入口。

## 适用场景

用于管理邮件连接、账号和平台通知所需的邮件传输能力。
