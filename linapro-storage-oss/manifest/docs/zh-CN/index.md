# 对象存储-阿里云 OSS

本文档随插件清单打包，面向插件市场展示。新增文档资源不会改变插件运行时行为。

## 概述

为宿主 Storage 领域能力提供阿里云 OSS 对象存储后端。

## 文档导航

- [配置说明](configuration.md)
- [更新日志](changelog.md)

## 功能亮点

- 为宿主 Storage 领域能力提供阿里云 OSS 对象存储后端。
- 提供阿里云 OSS等工作台入口。

## 适用场景

将本插件作为宿主`Storage()`领域能力背后的对象存储提供方。
