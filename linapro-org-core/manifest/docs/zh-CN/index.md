# 组织管理

本文档随插件清单打包，面向插件市场展示。新增文档资源不会改变插件运行时行为。

## 概述

提供部门与岗位管理的官方源码插件

## 文档导航

- [配置说明](configuration.md)
- [更新日志](changelog.md)

## 功能亮点

- 提供部门与岗位管理的官方源码插件
- 提供部门管理、岗位管理等工作台入口。
- 支持租户感知使用场景。

## 适用场景

当租户感知业务需要部门、岗位和用户组织归属时使用本插件。
